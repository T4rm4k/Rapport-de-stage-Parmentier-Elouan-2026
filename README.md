Pour la bonne compréhension du rapport, il peut être utile de se renseigner sur les liens suivants.

Tout d'abord les tutoriels vidéos de Leslie Lamport on été mon point d'entrée dans le langage TLA+: https://lamport.azurewebsites.net/tla/learning.html

Tous les fichiers commençant par "spec" correspondent aux spécifications des différents problèmes abordés dans le rapport. Je recommande en particulier d'avoir 
la spécification de EWD687a sous les yeux lors de la lecture du paragraphe 4.1.

Pour l'énoncé du problème Dijkstra-Scholten, il existe la version originale incomplète, jointe au format pdf. Et une version complète sur laquelle je me suis 
appuyée: https://www.cs.utexas.edu/~EWD/transcriptions/EWD06xx/EWD687a.html  
Je vous joint également une vidéo qui peut aider à la réprésentation de l'algorithme. (animation EWD687a.mp4)

apparaitres mes contributions d'ici la fin du stage.
